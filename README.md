# Eye-blink-detection
This is a ml project which differentiates fake videos and original videos by detecting eye blinking of a person in the video.
## Requirements
### Numpy : 
``` pip install numpy ```
### Pandas : 
```pip install pandas```
### CV2
``` pip install cv2 ```
### Cmake
``` pip install cmake ```
### Dlib
``` pip install dlib ```
### Scipy
``` pip install scipy ```
